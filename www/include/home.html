Welcome to CRISPR-SE (Search Engine)!

CRISPR-SE is a high-performance CRISPR search engine implemented with a brute force algorithm, where the off-targets search is complete/guaranteed.

We have pre-computed genome-wide gRNAs libraries for human and mouse genomes(online/offline) and other genomes for download(offline):

Pre-computed gRNA libraries

Online

  1. Find the position of your sequence/genes using BLAT or UCSC genome browser. (optional)
  2. View the pre-computed gRNAs within the positions

Offline

  1. Download the pre-computed bam files
  2. Search gRNAs using samtools

Build gRNA libraries

To build gRNA libraries for your own reference genomes.

  1. Download & compile CRISPR-SE from source code or the binary program
  2. Build index files for the reference genome
  3. Build index files for the region of interests, inputs are either in Fasta format or simple format(one gRNA per line). The simple format can be created using online scan tools. (skip this step for genome-wide gRNA design)
  4. Run CRISPR-SE for off-target search.
  5. Convert outputs into bam format and search gRNAs using samtools.

The source code is available at https://github.com/bil022/CRISPR-SE.
