Welcome to CRISPR-SE (Search Engine)!

CRISPR-SE is a high-performance CRISPR search engine implemented with a brute force algorithm, where the off-targets search is complete/guaranteed.

Pre-computed gRNA libraries

Online tools

  1. Find the position of your sequence/genes using BLAT or UCSC genome browser.
  2. Scan potential gRNAs (without off-targets search) online
  3. View the pre-computed gRNAs within the positions

Offline

  1. Software usage
  2. Download the pre-computed bam files
  3. Search gRNAs using samtools

Source code

You can download the source code from https://github.com/bil022/CRISPR-SE.
